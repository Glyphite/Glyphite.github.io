折腾三日

甲辰年 08月10日

为了制作这个博客，我足足折腾了三天，前面都还好，就是那个 npm install hexo-deployer-git --save 卡了我半天，期间也找过一些解决办法，说用淘宝镜像什么的，但是都没用，只能是一直反复尝试……实际上，我真正操作的时间只有不到10分钟，剩下的时间都在下载、失败、重试、下载、失败。哪怕是搭了梯子也没用，不知道是不是我操作问题，好在终于是成功了。

反正，仅在此给后来者一点建议：多试就完事了，下载到第三天的时候我自己都不报什么期望了，但是你看，这不就成功了？

坚持总是会带来一点惊喜的。

搭建框架的教程

分享一下使用Hexo框架搭建博客的相关教程：

https://tech.yemengstar.com/hexo-tutorial-deploy-githubpages-beginner/ #很详细的Hexo部署教程，基本每一步都有截图。

https://mao-design.pages.dev/posts/1185/ #以Github为载体，用 Netlify 部署博客的教程，可以提高访问速度。

框架搭建好之后就可以使用Hexo框架了。和直接使用HTML编写网页相比，Hexo的优势主要是方便快捷，添加文章不需要再敲代码，可以直接将Markdown语言渲染为网页。

给博客更换主题

进入hexo官网，选择主题，选择自己喜欢的主题然后下载即可（注：有些主题会下载出错，这种一般是作者的仓库没设置好，没辙，只能换个其他主题）。

有兴趣的话，你也可以考虑我现在正在用的主题 Cards 。点击就能跳转到对应的 Github 仓库，里面详细介绍了如何下载，使用教程等，具体更多的个人自定义的话，就得看你的前端基础了，稍微能看懂点代码就能自己修改。

一般安装主题的基本流程：

在自己博客的根目录下，把下载好的主题包 解压 并移到一个名为 themes 的文件夹下。

然后在回到根目录下，打开 _config.yml 配置文件，找到 theme 选项，输入你要使用的主题名字，注意选项与名字之间要有空格，例：theme: 主题名称

然后保存即可，在回到根目录上，打开终端，输入命令 hexo clean ，在输入命令 hexo g ，最后输入 hexo s ，输入这三个命令之后没有红色的字体就说明没有报错，此时终端会显示一串网址 http://localhost:4000 ，输入这串网址即可预览自己博客的主题了，如果想停止访问的话，在终端按下快捷键 ctrl+c 即可退出。
